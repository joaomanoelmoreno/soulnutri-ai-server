<analysis>**original_problem_statement:**
O usuário deseja construir o **SoulNutri**, um aplicativo que funciona como um agente de nutrição virtual ou Waze da alimentação. A principal funcionalidade é a identificação de pratos em tempo real por meio de imagens, fornecendo ao consumidor informações detalhadas (ingredientes, alérgenos, dados nutricionais, fatos científicos relevantes) antes da escolha do prato.

**PRODUCT REQUIREMENTS (baseado nos documentos e na conversa):**
1.  **Visão:** O slogan é SOULNUTRI - O SEU AGENTE DE NUTRICAO VIRTUAL. O app deve agir como um radar do prato, focando em segurança alimentar e informação de valor, atuando *antes* do consumo.
2.  **Premissa Fundamental:** **100% de precisão no reconhecimento** é a base de tudo. O erro no reconhecimento é veneno do app.
3.  **Arquitetura de Reconhecimento (Híbrida):** Um sistema em cascata foi aprovado para garantir a precisão.
    *   **Nível 1 (Índice Local):** OpenCLIP para pratos de parceiros (ex: Cibi Sana), com altíssima confiança.
    *   **Nível 2 (API Especializada):** Uma API de reconhecimento de alimentos com um generoso plano gratuito.
    *   **Nível 3 (IA Genérica):** Uma IA de visão de uso geral (como Gemini Vision) como fallback final.
4.  **Qualidade da Informação:**
    *   As informações devem ser educativas, cientificamente embasadas e despertar curiosidade, evitando dados óbvios.
    *   Deve haver um equilíbrio entre notícias boas e alertas, para não ser um app do apocalipse. Para alertas de risco, deve haver um link para uma fonte relevante.
5.  **UX/UI:**
    *   Interface com o mínimo de cliques, captura por toque na tela.
    *   Sistema de feedback para o usuário: confirmar, corrigir, cadastrar novo prato, descartar foto.
    *   Implementar um botão de Compartilhar para as curiosidades científicas.
6.  **Versão Premium:**
    *   Funcionalidades futuras incluem um contador nutricional que monitora o consumo do usuário com base em seu perfil (restrições, alergias, etc.), um histórico de pratos identificados e alertas personalizados no momento do consumo.
